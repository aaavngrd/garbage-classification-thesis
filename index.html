Система класифікації сміття на основі машинного навчання

Garbage Classification System Based on Machine Learning

Бакалаврська робота · 2026

Короткий опис

Система автоматичної класифікації зображень побутових відходів на основі згорткової нейронної мережі (PyTorch) з REST API інтерфейсом (FastAPI). Розпізнає 5 категорій: папір, пластик, скло, метал, органіка.

Класифікація відходів Machine Learning PyTorch FastAPI Transfer Learning Computer Vision REST API torchvision

Актуальність теми

Неправильне сортування побутових відходів залишається однією з ключових екологічних проблем сучасності. Більшість систем сортування на сміттєзвалищах виконуються вручну — це повільно, дорого та залежить від людського фактору.

Автоматизована система класифікації на основі нейронних мереж дозволяє значно пришвидшити та здешевити процес сортування, підвищити точність розпізнавання і знизити навантаження на операторів.

Проєкт надає REST API який можна інтегрувати у виробничі конвеєрні системи або мобільні застосунки для підтримки роздільного збору відходів громадянами.

Мета та основні завдання

Мета дослідження: розробити систему автоматичної класифікації зображень побутових відходів на основі глибокого навчання з REST API інтерфейсом для інтеграції у реальні застосунки.

Основні завдання

  1. Збір та підготовка датасету. Формування збалансованого набору зображень відходів (папір, пластик, скло, метал, органіка) з поділом на навчальну, валідаційну та тестову вибірки.
  2. Навчання класифікаційної моделі. Розробка та навчання CNN на базі PyTorch із застосуванням transfer learning для досягнення високої точності класифікації.
  3. Розробка REST API. Реалізація FastAPI сервера для HTTP-доступу до моделі з Swagger документацією та підтримкою завантаження зображень.
  4. Тестування та оцінка якості. Оцінка метрик точності та F1-score на тестовій вибірці.
  5. Документування. Технічна документація та інструкції з розгортання відповідно до стратегії Release Flow.

Методологія дослідження

  • Transfer Learning

    Використання попередньо навченої моделі з torchvision як базової архітектури з подальшим донавчанням на цільовому датасеті відходів.

  • Data Augmentation

    Трансформації зображень (поворот, масштабування, зміна яскравості) для збільшення різноманітності навчальних даних та покращення узагальнення.

  • FastAPI Backend

    Асинхронний Python сервер з автоматичною Swagger документацією та підтримкою multipart/form-data для завантаження зображень.

  • Release Flow (Git)

    Стратегія гілкування: короткі feature-гілки від main, явні release-гілки з семантичними тегами версій (v1.0.0, v1.1.0).

Очікувані результати

  • Навчена нейронна мережа для класифікації відходів на 5 категорій з достатньою точністю для практичного застосування.
  • REST API на FastAPI з Swagger документацією, готовий до інтеграції у сторонні застосунки та системи.
  • Публічний Git-репозиторій з документованим процесом розробки згідно Release Flow з тегованими релізами.
  • Практична основа для розвитку у напрямку мобільного застосунку або IoT-інтеграції для розумного сортування відходів.
